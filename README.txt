Requires Microsoft Visual C++ Redistributable for Visual Studio 2017: https://aka.ms/vs/15/release/vc_redist.x64.exe

Keyboard Shortcuts
------------------
Ctrl+N:       New document
Ctrl+S:       Save
Ctrl+Shift+S: Save As
Ctrl+O:       Open
Ctrl+Z:       Undo
Ctrl+Shift+Z: Save As
F1:           Switch to Select tool (select / move / delete points)
  Delete      Delete selected point
F2:           Switch to Add Point tool (adding points to walls)
F3:           Switch to Wall tool (drawing walls)
  Backspace:  Cancel last point
  Escape:     Cancel all points
